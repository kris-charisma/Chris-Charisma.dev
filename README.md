# Chris Charisma — Portfolio

A modern, bilingual (EN/FR) software engineer and web designer portfolio. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools.

## Features

- **Bilingual** — Full English and French translations via a custom i18n system
- **Dark glassmorphism UI** — Premium glass-card design with smooth hover interactions
- **Project showcase** — 11 main projects + 5 sample projects with live previews
- **Skills section** — Frontend, backend, database, design, and more
- **Pricing & services** — Transparent service tiers and offerings
- **Contact form** — WhatsApp and email integration
- **SEO-friendly** — Semantic HTML, meta tags, and accessibility attributes
- **Fully responsive** — Mobile, tablet, and desktop layouts
- **Scroll animations** — Intersection Observer-based reveal animations
- **Performance optimized** — Lazy-loaded images, minimal dependencies

## Setup

No build tools required. Simply open `index.html` in your browser:

```bash
# Clone the repo
git clone https://github.com/kris-charisma/Chris-Charisma.dev.git

# Open in browser
open index.html
# or
start index.html   # Windows
```

### Local development

Use any static file server for local development:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```
